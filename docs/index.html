Motivation

Modern 3D animation software like Blender, Autodesk Maya and Houdini offer a wide range of tools used for creating 3d and 2d animations. This includes simulations, special effects, modelling and shading. BlendAIR aims to recreate the basic animation pipeline in a lightweight, browser based tool. As the name implies this project is heavily inspired by the open source project Blender.

There are multiple reasons for using the web platform of native.

  1. Web applications work on almost any device without the need for platform specific code. This includes mobile, although it may not be used for content creation, there is potential for usage in presenting.
  2. No need for local installation of the tool.
  3. Since this is a project for my university's multimedia class I aimed to combine it with the web engineering project from another class.

Features

The main objective of this project is to create a Blender like 3d tool, the following features is what I achieved until now.

  • interactive, resizable windows which can be used to implement various Blender interfaces. For example:
    • viewport: selection and transformation of entities, orbit and panning camera
    • timeline: animation scrolling
  • rendering support for mesh type entities
  • groundwork for keyframe based animations (no UI yet)
  • importable .obj models

Viewport Controls

left mouse button: lmb
middle mouse button: mmb

camera rotation: lmb drag + LeftAlt or mmb drag
camera panning: lmb drag + LeftAlt + LeftShift or mmb drag + LeftShift
camera zoom: mmb scroll or mmb scroll + LeftShift (for fine control)

object move: KeyG use lbm click to finalize
object scale: KeyS use lbm click to finalize
object rotate: KeyR use lbm click to finalize

select: lmb click or lmb click + LeftShift (for multiple)
abort: Escape
